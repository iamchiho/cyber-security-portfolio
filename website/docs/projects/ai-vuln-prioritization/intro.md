# Welcome to AI Vulnerability Prioritization

This project aims to enhance vulnerability management by combining automated scanning with AI-driven risk prioritization. Traditional vulnerability scanning tools can generate large volumes of findings, making it challenging for security teams to quickly identify and prioritize the most critical risks.

To address this, this project integrates vulnerability scanning results with intelligent analysis to produce actionable, prioritized insights.

---

# Project Objective

The ultimate goal of this project is to bridge the gap between vulnerability detection and risk-based decision-making by integrating automated scanning, data processing, and AI-assisted analysis into a streamlined workflow.

---

# Tools are leveraged

**OpenVAS (Greenbone)**

Greenbone Networks develops and maintains Open Vulnerability Assessment Scanner (OpenVAS), a full-featured, open-source vulnerability scanning solution.

OpenVAS is designed to scan systems, networks, and applications for security weaknesses, including:
	•	Outdated software
	•	Misconfigurations
	•	Weak authentication settings
	•	Known CVEs and exposed services

It provides continuously updated vulnerability tests, detailed scan reports, and a web-based management interface. In this project, OpenVAS serves as the primary data source for identifying security findings across target environments.


**Python**

Python is used to process and analyze the scan results generated by OpenVAS.

Within this project, Python is responsible for:
+ Parsing and cleaning exported scan data (CSV/XML)
+ Enriching findings with asset criticality and contextual information
+ Calculating weighted risk scores
+ Preparing structured data for AI-driven analysis
+ Generating final prioritized reports

Python acts as the automation and integration layer between vulnerability scanning and AI analysis.


**OpenAI (LLM Integration)**

OpenAI provides the Large Language Model (LLM) used in this project.

The LLM is applied to:
+ Analyze vulnerability context beyond raw CVSS scores
+ Generate composite risk assessments
+ Produce human-readable summaries
+ Recommend remediation priority levels

By combining structured scan data with contextual AI reasoning, the project moves beyond static scoring models and delivers more intelligent, business-aware risk prioritization.
